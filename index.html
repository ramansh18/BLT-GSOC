Google Summer of Code

Google Summer of Code with Open Worldwide Application Security Project (OWASP) Foundation

The Google Summer of Code program (GSoC) encourages student participation in open source development. Accepted students are paired with OWASP mentors who guide implementation across real security-focused deliverables. OWASP BLT participates in GSoC through the OWASP Foundation, connecting students with real-world open-source security work since 2020.

This program runs completely online. Students and mentors from more than 100 countries have participated over past years.

OWASP BLT & GSOC Partnership

OWASP is an open community dedicated to enabling organizations to conceive, develop, acquire, operate, and maintain applications that can be trusted. OWASP has been an active participant in the Google Summer of Code program, providing students with opportunities to contribute to multiple OWASP projects.

OWASP BLT (Bug Logging Tool) has participated in Google Summer of Code for 5 consecutive years, from 2020 to 2024. BLT is an open-source platform focused on making the internet safer through community-driven bug reporting and collaboration.

All students currently enrolled in an accredited institution are welcome to participate in GSoC, hopefully along with the OWASP Foundation.
